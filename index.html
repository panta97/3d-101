The Math of 3D Engines, Hands On

For developers who can ship software but glaze over at "orthonormal basis."

Every 3D engine — Unreal, Unity, the renderer inside your favorite game — is built from a surprisingly small set of mathematical ideas. This course teaches them the way you learned everything else: by implementing them.

Each module is a sequence of short reads, live diagrams you can grab and drag, and in-browser coding exercises. You write the math functions — dot, cross, normalize, matrix multiply — and your code immediately drives the visualizations on the page. By the later modules, the renderer drawing the diagrams is made of functions you wrote.

No frameworks, no black-box math library. Where we must use something you haven't built yet (like 3D projection, in early modules), we say so explicitly and promise the module where you'll build it yourself.
